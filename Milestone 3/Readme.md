## Overview

ClauseAI is a multi-agent AI system for analyzing contracts across **Legal, Compliance, Finance, and Operations** domains. It uses **RAG + LangGraph** to enable structured reasoning, shared memory, and coordinated agent execution.

---

## Features

* Contract ingestion & clause chunking
* Semantic embeddings and vector retrieval
* Domain-specific analysis agents
* Shared agent memory & cross-validation
* Sequential and parallel execution

---

## Tech Stack

* Python
* LangGraph & LangChain
* Pinecone (Vector DB)
* Hugging Face / Gemma models
* Jupyter Notebook

---

## Milestones

* **Milestone 1:** Planning & RAG setup
* **Milestone 2:** Multi-agent pipelines & orchestration
* **Milestone 3:** Parallel execution & memory persistence
